This section is dedicated to providing you with the most recent updates and essential information.


## Active Announcements[​](https://upstox.com/developer/api-documentation/announcements/#active-announcements "Direct link to heading")
  * **December 6, 2025:[NSE’s Pre-Open Session for Futures LIVE from 8 December](https://upstox.com/developer/api-documentation/announcements/nse-pre-open-session)**
NSE's pre-open session for futures is now live from December 8, 2025. This session allows traders to place and modify orders before the regular trading hours begin, providing an opportunity to react to overnight news and market developments.
For more details, please refer to the [NSE’s Pre-Open Session for Futures LIVE from 8 December](https://upstox.com/developer/api-documentation/announcements/nse-pre-open-session).
  * **August 22, 2025:[V2 Websocket Discontinued](https://upstox.com/developer/api-documentation/announcements/v2-websocket-discontinued)** Discontinued
The Market Data Feeder V2 Websocket service will become discontinued on August 22, 2025, and will be completely stopped after this date. All users should migrate to the V3 Websocket service to avoid any disruption in market data streaming.
For more details, please refer to the [V2 Websocket Discontinued](https://upstox.com/developer/api-documentation/announcements/v2-websocket-discontinued).
  * **July 19, 2025:[Fund and Margin API Response Change](https://upstox.com/developer/api-documentation/announcements/fund-margin-api-change)**
Starting July 19, 2025, the [Fund and Margin API](https://upstox.com/developer/api-documentation/get-user-fund-margin) will return combined funds for both Equity and Commodity segments in the `equity` object. This change affects applications that process segment specific fund data separately.
For more details, please refer to [Fund and Margin API Response Change](https://upstox.com/developer/api-documentation/announcements/fund-margin-api-change)
  * **Jun 30, 2025:[Launch of Trailing Stop Loss order](https://upstox.com/developer/api-documentation/announcements/tsl-gtt-order)** Beta
We've introduced Trailing Stop Loss functionality to our GTT Order API, allowing your stop-loss orders to automatically adjust as the market moves in your favor, providing dynamic risk management.
For more details, please refer to the [Launch of Trailing Stop Loss order](https://upstox.com/developer/api-documentation/announcements/tsl-gtt-order).
  * **June 30, 2025:[Deprecation of several V2 APIs](https://upstox.com/developer/api-documentation/announcements/deprecation-notice-v2)** Deprecated
We are deprecating several v2 APIs to streamline our offerings and encourage users to migrate to the corresponding v3 endpoints. For more details, please refer to the [Deprecation Notice](https://upstox.com/developer/api-documentation/announcements/deprecation-notice-v2).
  * **Jun 20, 2025:[Support for MTF in GTT Order](https://upstox.com/developer/api-documentation/announcements/gtt-mtf-support)**
We have enabled support for the Margin Trading Facility (MTF) product within the GTT order placement API. This enables the placement of GTT orders with margin funding directly through the API.
For more details, please refer to the [Support for MTF in GTT Order](https://upstox.com/developer/api-documentation/announcements/gtt-mtf-support).
  * **May 13, 2025:[Expired Instruments APIs Launch](https://upstox.com/developer/api-documentation/announcements/expired-instruments-api)** ![](https://upstox.com/developer/api-documentation/announcements/)
We are rolling out new Expired Instruments APIs under the Upstox Plus plan. This suite of APIs provides access to historical data for expired instruments, offering the following features:
    * Get Expiries.
    * Get Expired Option Contracts.
    * Get Expired Future Contracts.
    * Get Expired Historical Candle Data.
For more details, please refer to the [Expired Instruments API Launch](https://upstox.com/developer/api-documentation/announcements/expired-instruments-api).
  * **May 13, 2025:[Websocket Plus Features](https://upstox.com/developer/api-documentation/announcements/websocket-plus)** ![](https://upstox.com/developer/api-documentation/announcements/)
We are rolling out new websocket features under the Upstox Plus plan, enhancing our v3 websocket with:
    * Increased concurrent connections (up to 5 per user).
    * Full D30 subscription mode supporting up to 50 instruments per connection.
For more details, please refer to the [Websocket Plus Features](https://upstox.com/developer/api-documentation/announcements/websocket-plus).
  * **Apr 17, 2025:[Enhanced Historical Candle Data APIs - V3 Launch](https://upstox.com/developer/api-documentation/announcements/enhanced-historical-candle-data-apis-v3)**
We're excited to introduce enhanced V3 versions of both standard [Intraday](https://upstox.com/developer/api-documentation/get-intra-day-candle-data) and [Historical](https://upstox.com/developer/api-documentation/get-historical-candle-data) Candle Data APIs with expanded intervals and improved flexibility. These APIs support customizable time units from `minutes` to `months` with flexible intervals, offering more granular control for technical analysis.
For more details, please refer to the [Enhanced Historical Candle Data APIs - V3 Launch](https://upstox.com/developer/api-documentation/announcements/enhanced-historical-candle-data-apis-v3).
