# Orders
This guide describes how to manage orders using the SDK. You can create, modify, cancel, and retrieve order details for equities, derivatives, and commodities.
## [Place Order](https://groww.in/trade-api/docs/python-sdk/orders#place-order)
Use the place_order SDK method to register a new order. When successful, the method returns a unique order ID and its initial status.
### [Python SDK Usage](https://groww.in/trade-api/docs/python-sdk/orders#python-sdk-usage)
```
from growwapi import GrowwAPI

# Groww API Credentials (Replace with your actual credentials)
API_AUTH_TOKEN = "your_token"

# Initialize Groww API
groww = GrowwAPI(API_AUTH_TOKEN)

place_order_response = groww.place_order(
    trading_symbol="WIPRO",
    quantity=1,
    validity=groww.VALIDITY_DAY,
    exchange=groww.EXCHANGE_NSE,
    segment=groww.SEGMENT_CASH,
    product=groww.PRODUCT_CNC,
    order_type=groww.ORDER_TYPE_LIMIT, # You can also use ORDER_TYPE_MARKET, ORDER_TYPE_STOP_LOSS, ORDER_TYPE_STOP_LOSS_MARKET for different order types
    transaction_type=groww.TRANSACTION_TYPE_BUY,
    price=250,               # Optional: Price of the stock (for Limit orders)
    trigger_price=245,       # Optional: Trigger price (if applicable)
    order_reference_id="Ab-654321234-1628190"  # Optional: User provided 8 to 20 length alphanumeric reference ID to track the order
)
print(place_order_response)
```

#### [Request Schema](https://groww.in/trade-api/docs/python-sdk/orders#request-schema)
Name | Type | Description
---|---|---
trading_symbol `*` | string | Trading Symbol of the instrument as defined by the exchange

quantity `*` | int | Quantity of instrument to order

price | float | Price of the instrument in rupees case of Limit order

trigger_price | float | Trigger price in rupees for the order

validity `*` | string |  [Validity](https://groww.in/trade-api/docs/python-sdk/annexures#validity) of the order

exchange `*` | string |  [Stock exchange](https://groww.in/trade-api/docs/python-sdk/annexures#exchange)

segment `*` | string |  [Segment](https://groww.in/trade-api/docs/python-sdk/annexures#segment) of the instrument such as CASH, FNO etc.
product `*` | string |  [Product type](https://groww.in/trade-api/docs/python-sdk/annexures#product)

order_type `*` | string |  [Order type](https://groww.in/trade-api/docs/python-sdk/annexures#order-type)

transaction_type `*` | string |  [Transaction Type](https://groww.in/trade-api/docs/python-sdk/annexures#transaction-type) of the trade

order_reference_id | string | User provided 8 to 20 length alphanumeric string with atmost two hyphens. (-)

`*`required parameters

## [Get Trades for Order](https://groww.in/trade-api/docs/python-sdk/orders#get-trades-for-order)
Retrieve details of trades executed for a particular order using this `get_trade_list_for_order` method. Maximum `page_size` is 50.
### [Python SDK Usage](https://groww.in/trade-api/docs/python-sdk/orders#python-sdk-usage-3)
```
from growwapi import GrowwAPI

# Groww API Credentials (Replace with your actual credentials)
API_AUTH_TOKEN = "your_token"

# Initialize Groww API
groww = GrowwAPI(API_AUTH_TOKEN)

trade_list_response = groww.get_trade_list_for_order(
    groww_order_id="GMK39038RDT490CCVRO",
    segment=groww.SEGMENT_CASH,
    page=0, # Optional: Page number for paginated results
    page_size=50 # Optional: Number of trades to fetch per page (default is 50)
)
print(trade_list_response)
```

#### [Request Schema](https://groww.in/trade-api/docs/python-sdk/orders#request-schema-3)
Name | Type | Description
---|---|---
groww_order_id `*` | string | Order id generated by Groww for an order

segment `*` | string |  [Segment](https://groww.in/trade-api/docs/python-sdk/annexures#segment) of the instrument such as CASH, FNO etc.
page `*` | int | The page number for paginated results, used to fetch trade details in multiple pages if there are many trades associated with the order.

page_size `*` | int | The number of trades to fetch per page. Maximum is 50. If not provided, it will return the default 50 number of trades.

`*`required parameters
